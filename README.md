Smart Shop Management System
A complete web-based shop management system designed to simplify daily shop operations like inventory tracking, billing, quotations, worker management, and AI-based insights.

 Live Features
1) Dashboard
- View shop performance
- Track sales, stock, and activity
2) Inventory Management
- Add / Update / Delete products
- Track quantity, price, expiry
- Automatic stock update after billing
3) Billing System
- Create customer bills
- GST calculation (18%)
- Auto stock deduction
- WhatsApp bill sharing
- QR code for UPI payment
4) Quotation System
- Create and manage quotations
- Convert quotations into bills easily
5) Worker Management
- Add workers
- Track salary
- Attendance management
6) Smart Offer System
- Create offers with:
  - Offer Name
  - Discount %
  - Minimum Purchase
  - Valid Dates
- Auto discount applied in:
  - Billing
  - Quotations
- Attractive offer cards on dashboard
7) Settings
- Shop name, address, phone
- Offer configuration
- Organized UI for better usability
8) AI Shop Assistant (Gemini AI)
- Ask questions like:
  - “What is my total sales?”
  - “Which product is low in stock?”
  - “Show current offers”
- Uses real-time Firebase data

Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Firebase (Firestore)
- AI Integration: Google Gemini API
- Hosting: Firebase Hosting / GitHub
